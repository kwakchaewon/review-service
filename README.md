## 🎮 서비스 소개
Spring boot, REST API, JPA 기반 맛집 리뷰 서비스 <br/>
<br/>

## 🛠 사용기술
### **Backend**
- Spring boot, Spring security, REST API, JPA

### **Infra**
- AWS EC2, Docker, MYSQL
<br/>

## ➡️ 사용자 Flow
- 회원가입 후 로그인 할 수 있다.
- 맛집을 등록할 수 있다.
- 맛집을 삭제할 수 있다.
- 맛집별 리뷰를 작성할 수 있다.
- 맛집에 작성한 리뷰를 삭제할 수 있다.
- 맛집에 작성된 리뷰와 별점을 확인할 수 있다.
